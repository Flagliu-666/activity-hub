# 活动互助站 - 免费部署指南

## 方案一：Cloudflare Pages（推荐 ⭐⭐⭐⭐⭐）

### 优点
- ✅ 完全免费
- ✅ 国内可访问
- ✅ 自动 HTTPS
- ✅ 不需要服务器
- ✅ 电脑关机也不影响

### 步骤

#### 1. 注册 Cloudflare（2 分钟）
1. 打开 https://dash.cloudflare.com/sign-up
2. 用邮箱注册（免费）
3. 验证邮箱

#### 2. 创建 Pages 项目
1. 登录后点击 **"Create a project"** → **"Pages"**
2. 选择 **"Direct upload"**（直接上传）
3. 下载 Cloudflare CLI：
   ```powershell
   npm install -g @cloudflare/wrangler
   wrangler login
   ```
4. 在项目目录 `activity-hub` 下创建 `wrangler.toml` 文件：
   ```toml
   name = "activity-hub"
   type = "javascript"
   account_id = ""
   workers_dev = true
   route = ""
   zone_id = ""
   
   [site]
   bucket = "./public"
   ```
5. 上传静态文件：
   ```powershell
   cd c:\Users\31765\CodeBuddy\Claw\activity-hub
   npx wrangler pages deploy public --project-name activity-hub
   ```
6. 部署完成后会给你一个域名，如 `activity-hub.pages.dev`

#### 3. 后端部署（需要云函数）
由于项目有 Node.js 后端（Express + SQLite），需要额外部署后端：

**选项 A：Cloudflare Workers**（免费额度）
- 将 `server.js` 改造为 Workers 格式
- 使用 Cloudflare D1（SQLite 数据库）

**选项 B：Railway.app**（免费额度）
- 打开 https://railway.app
- 注册账号
- 新建项目 → Deploy from GitHub Repo
- 连接你的代码仓库
- 自动检测 Node.js 并部署

## 方案二：Railway.app（最简单 ⭐⭐⭐⭐）

### 优点
- ✅ 免费额度（5 美元/月）
- ✅ 一键部署
- ✅ 自动 HTTPS
- ✅ 国内可访问

### 步骤

1. 打开 https://railway.app
2. 用 GitHub 账号登录
3. 点击 **"New Project"** → **"Deploy from GitHub repo"**
4. 选择 `activity-hub` 仓库
5. Railway 会自动检测 Node.js 并部署
6. 部署完成后获得一个公网 URL

### 配置环境变量
在 Railway 后台添加环境变量：
```
PORT=3000
```

## 方案三：Render.com（免费 ⭐⭐⭐）

### 优点
- ✅ 完全免费
- ✅ 自动 HTTPS
- ⚠️ 国内访问可能稍慢

### 步骤

1. 打开 https://render.com
2. 用 GitHub 账号登录
3. 点击 **"New Web Service"**
4. 连接你的 GitHub 仓库
5. 配置：
   - **Name**: activity-hub
   - **Environment**: Node
   - **Build Command**: `npm install`
   - **Start Command**: `node server.js`
6. 点击 **"Create Web Service"**

## 方案四：本地内网穿透（最快，但电脑不能关）

### 使用 ngrok

1. 下载 ngrok：https://ngrok.com/download
2. 注册账号并获取 Token
3. 运行：
   ```powershell
   ngrok http 3000
   ```
4. 获得一个公网 URL，如 `https://xxxx.ngrok.io`

### 使用 localtunnel（无需注册）

```powershell
npx localtunnel --port 3000
```
